# Portfolio

## Quick start

Quick start options:

- Clone the repo: `git clone https://github.com/kumaratulbatham/portfolio.git`.
- [Download from Github](https://github.com/kumaratulbatham/portfolio.git).
- `npm install`

- `npm start`

Go to Github Pages: <https://kumaratulbatham.github.io/portfolio/>
